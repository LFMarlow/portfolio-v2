04

Chronologie

Progression du projet infrastructure EPSI.

Phase 1

Fondations

Mise en place du node Proxmox sur bare-metal OVH, configuration des VLANs (CORE, SERVER, DMZ/ADMIN, LAN), déploiement de pfSense comme gateway centrale.

Phase 2

Services Core

Déploiement MariaDB master/slave, serveurs LAMP, Nginx reverse proxy. Mise en place de la segmentation réseau inter-VLAN.

Phase 3

Sécurité & Monitoring

Installation Wazuh SIEM + agents OSSEC, Suricata IDS/IPS, Zabbix monitoring, GLPI gestion de parc. Couverture complète de l'infrastructure.

Phase 4

Bastion & Hardening

Apache Guacamole comme bastion SSH/RDP centralisé. Hardening Lynis (65→82), auditd, fail2ban, politiques SSH, snapshots systématiques.

Phase 5

Automatisation

Terraform pour le provisioning automatisé des VMs LAMP, configuration Nginx dynamique, versioning Git/GitHub. Documentation complète.
